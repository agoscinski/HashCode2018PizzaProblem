# HashCode2018PizzaProblem
A visualization script to visualize the problem and solution for the practice problem of the Googles Hash Code 2018 https://hashcode.withgoogle.com/

### Requirements:
- Numpy 
- Matplotlib

### Usage:
```
python <dataset>
```
